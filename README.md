📝 To-Do List Web App

A simple and interactive To-Do List built using HTML, CSS, and Vanilla JavaScript.
Users can add tasks, mark them complete with a checkbox, and delete tasks instantly—without reloading the page.

🚀 Features:
-Add new tasks
-Mark tasks as completed
-Delete tasks
-Keyboard support (Press Enter to add)
-Clean and responsive UI
🛠 Technologies Used:
-HTML5
-CSS3
-JavaScript (Vanilla JS)
📂 How to Use:
-Open index.html in any browser.
-Type a task → Click Add Task or press Enter.
-Tick checkbox to complete.
-Click Delete to remove a task.
